1. Install k3d, kubectl, helm CLIs
2. run: k3d cluster create gecko2-cluster --port "8888:80@loadbalancer"
3. K3d cluster runs inside its own isolated Docker container...
k3d image import gecko2/user-service:latest gecko2/mobile-bff:latest gecko2/messaging-service:latest gecko2/message-persistence-service:latest gecko2/media-service:latest gecko2/api-gateway:latest -c gecko2-cluster
4. we run / install: helm upgrade --install user-service .
